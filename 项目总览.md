# 日历应用项目总览

## 📋 项目简介

这是一个基于Flutter开发的日历应用，符合RFC5545标准，支持日程管理、提醒功能，并具备扩展能力。

## 🎯 核心功能

### 基本功能（必须实现）
- ✅ 日历视图展示（月视图、周视图、日视图）
- ⏳ 日程添加、编辑、查看和删除
- ⏳ 日程提醒功能

### 扩展功能（可选实现）
- ⏳ 日历事件导入导出（iCal格式）
- ⏳ 网络订阅功能
- ⏳ 农历相关功能

## 🛠️ 技术栈

- **框架**: Flutter 3.9.2+
- **状态管理**: Riverpod 2.5.1
- **本地存储**: Hive 2.2.3
- **日历组件**: table_calendar 3.2.0
- **通知**: flutter_local_notifications 19.5.0
- **时区处理**: timezone 0.10.1

## 🚀 快速开始

